Steps to compile and run your application:
1. Ensure you are in the current folder/directory
2. Ensure you have psycopg2 installed on your environment
    2.1 If you do not have it installed, please run the command 'pip install psycopg2'
3. Set up the database in pgadmin4 using the SQL txt file with the corresponding credentials
4. run the main python file using the command 'python a3main.py'
